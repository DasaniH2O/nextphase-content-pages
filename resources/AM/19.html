Setup Guide · claude-seo (open source)

Run a full SEO audit from your terminal — one command, a squad of sub-agents, $0.

claude-seo is a free, MIT-licensed skill you install into Claude Code. One command fans out specialist sub-agents in parallel across a site — technical, content, schema, competitors — plus GEO/AEO checks (whether ChatGPT and Perplexity recommend the business). It returns a 0–100 health score and a prioritized fix list. Everything below runs on your own machine.

Install & run

# 1 — prerequisites: Python 3.10+ and the Claude Code CLI
python3 --version      # need 3.10 or newer
npm i -g @anthropic-ai/claude-code

# 2 — clone the skill
git clone https://github.com/AgriciDaniel/claude-seo.git

# 3 — inside Claude Code, run the audit against your site
claude
> /seo-audit https://yoursite.com

  • Bolt on the opt-in extensions (e.g. DataForSEO, Firecrawl) for live rankings & search volume — those may carry their own cost

The honest catch

The 0–100 health score is the model’s best judgment — not authoritative Google ranking data. Use it to prioritize fixes, not to report position. The “$5,000 agency audit in 4 minutes” framing is rhetoric, not a measured benchmark. And confirm the repo before trusting it: several similarly-named SEO skills exist.
